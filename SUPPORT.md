# Support

Use the following paths depending on what you need:

- usage questions: start with [README.md](./README.md) and the guides in [docs/](./docs/)
- bug reports: open a GitHub issue with the bug template
- feature ideas: open a GitHub issue with the feature template
- security concerns: follow [SECURITY.md](./SECURITY.md)

For converter-related problems, the most helpful reports include:

- a minimal markdown sample
- expected DOCX output
- actual DOCX output
- package version
- runtime details
