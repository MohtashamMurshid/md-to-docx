# Agent Skill Usage

This repository also includes a reusable skill for agent workflows.

## Install From GitHub

```bash
npx skills add MohtashamMurshid/md-to-docx --skill md-to-docx --agent cursor --yes --full-depth
```

## Install From A Local Clone

```bash
npx skills add . --skill md-to-docx --agent cursor --yes --full-depth
```

## Discover Available Skills

```bash
npx skills add MohtashamMurshid/md-to-docx --list --full-depth
```

## When To Use It

Use the skill when you want an agent to:

- convert Markdown strings or files to `.docx`
- configure styling, alignment, or fonts
- build multi-section Word documents
- work with TOC, page numbering, headers, and footers
